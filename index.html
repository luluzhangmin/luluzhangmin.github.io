Step 1. Create Repository Log in to https://github.com. Click the “+” in the upper right corner and select New repository. Enter the domain name under Repository name xx.github.io For example: luluzhangmin.gothub.io Select the Public and Add a README file options, and click Create repository. Click Settings in the navigation bar, select the master branch under Source, and click Save.
