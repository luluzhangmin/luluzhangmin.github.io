Deploy the Blog to GitHub Pages

Step 1. Create Repository

  1. Log in to https://github.com.

  2. Click the “+” in the upper right corner and select New repository.

  3. Enter the domain name under Repository name

    xx.github.io

       For example: luluzhangmin.gothub.io

  4. Select the Public and Add a README file options, and click Create repository.

  5. Click Settings in the navigation bar, select the master branch under Source, and click Save.

  6. Modify the local “config.toml file.”, Modify the value of baseUrl to the set domain name.

    For example: baseUrl = "https://luluzhangmin.github.io"

Step 2. Deploy the Blog to GitHub

  1. Execute the following command in the root directory of the site To generate a “public” directory.

    **hugo –theme=**theme name –baseUrl=“https://nickname.github.io/” –buildDrafts

    For example: hugo --theme=lulu --baseUrl="https://luluzhangmin.github.io/" --buildDrafts

  2. Execute the following command to enter the directory.

    cd public

  3. Execute the following command to initialize the file.

    git init

  4. Execute the following command to load all files.

    git add .

  5. Execute the following command to submit the information and add all the files to the local warehouse.

    git commit -m “custom submission instructions”

  6. Execute the following command to associate the local warehouse with the warehouse on GitHub.

    git remote add origin https://github.com/github username/domain.git

    For Example: git remote add origin https://github.com/luluzhangmin/luluzhangmin.github.io.git

  7. Execute the following command to pull the master branch.

    git pull origin master

  8. Execute the following command to push the master branch

    git push -u origin master

  9. If the first push fails, execute the following command to push the blog to the master branch.

    git push -u origin master -f

Step 3. Resolve io Domain Name

  1. Search for View Network Connection in the local PC search box, press Enter.
  2. Find WLAN and right click Properties.
  3. Select Internet Protocol Version 4 (TCP/IPv4) and click Properties.
  4. Select Use the DNS server address belowand enter “8.8.8.8” as Preferred DNS Server.
  5. Click Confirm and then click Close.

Step 4. Visit Blog

Enter the io address you created in the browser to access your own blog, For example, my blog address ishttps://luluzhangmin.github.io .
