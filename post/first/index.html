Build a Hugo Blog in Windows 10

Prepare the Environment

Step 1. Install and Configure Github

  1. Visit https://git-scm.com/downloads and download Git.
  2. Install Git, all options are selected by default.
  3. Register Git account and configure Key. Please Baidu or Goole for detailed operation.

Step 2. Install and Configure Hugo

Download the Hugo Tool

  1. Visit https://github.com/gohugoio/hugo/releases and download Git.
  2. Unzip the file package.

Configure environment variables for hugo

  1. Click the local PC and right-click, click Properties.
  2. Select “Advanced System Settings> Environment Variables”.
  3. Select “Path” under User Variables and click “Edit”.
  4. Click “New”. Create a new storage path for hugo, click “Ok”.
  5. Open the CMD, execute the command “hugo version”, the hugo version information is displayed, the tool is installed successfully.

Deploy a Hugo Blog Locally

  1. Enter the directory of the custom hugo name, here is the directory “E:\hugo” as an example.

  2. Right-click and select “Git Bash Here”.

  3. Execute the flllowing command to create a new blog file.

    hugo new site blogname

     For example: hugo new site myblog

  4. Execute the flllowing command to enter myblog file.

    cd themename

     For example: cd myblog

  5. Log in to https://themes.gohugo.io/tags/documentation/ to choose your favorite theme.

  6. Click “Download” to enter the “Code” page of github.

  7. Click the “Code” button and select copy under the “HTTPS” protocol.

  8. In the Git tool box, Execute the following command to download the blog theme to the local.

    git clone theme address theme/theme’s name

     For example: git clone https://github.com/gyorb/hugo-dusk.git themes/lulu

  9. Execute the following command to aggregate the pages.

    hugo new post/xx.md

     For example: hugo new post/first.md

  10. Use Markdown to edit the document, the document format is “xxx.md”, and then put the document in “post”.

  11. Execute the following command to start the blog locally and run it.

    hugo server -t theme’s name –buildDrafts

     For example: hugo server -t lulu --buildDrafts

  12. Enter the address “http://local:1313” to visit yourself blog, The information displayed here is the document information edited by yourself.
